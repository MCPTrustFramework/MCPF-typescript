✅ Full type safety with TypeScript 5.3+
✅ ES Modules (ESM) support
✅ DID/VC module (create, verify, resolve)
✅ ANS client (resolve, register agents)
✅ MCP Registry client (search, verify servers)
✅ A2A client (check delegation, policies)
✅ Example scripts
✅ GitHub Actions CI/CD
✅ ESLint + Prettier configuration

Modules:
✅ MCPF - Unified client
✅ DIDManager - DID operations
✅ VCIssuer - Credential issuance
✅ VCVerifier - Credential verification
✅ ANSClient - Agent Name Service
✅ MCPRegistry - MCP Trust Registry
✅ A2ARegistry - A2A Delegation

Zero Dependencies:
- Uses native fetch API
- No external runtime dependencies
- Only dev dependencies for tooling

Usage:
import { MCPF } from 'mcpf-typescript';

const mcpf = new MCPF();
const agent = await mcpf.ans.resolve({ name: 'agent.example.agent' });
const valid = await mcpf.did.verifyAgent(agent.did);
